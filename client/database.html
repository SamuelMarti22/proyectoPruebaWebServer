          ¿Por qué es Caso 4?
        

        

          Esta página solicita múltiples archivos distintos al servidor:
          el documento HTML, la hoja de estilos CSS, la API JSON (/api/movies)
          y el archivo TXT plano (/api/db).
          Esto contrasta con el Caso 3 que carga un solo recurso pesado.
          El servidor Node.js sirve todos estos recursos desde un único proceso HTTP.
