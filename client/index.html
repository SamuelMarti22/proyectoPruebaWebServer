Stack Técnico

Servidor:
Node.js nativo — sin frameworks.
HTTP module puro.
Cliente:
HTML + CSS + JS vanilla.
Sin librerías externas.
Base de datos:
Archivo database.txt
separado por pipes "|".
API:
GET /api/movies
POST /api/movies
